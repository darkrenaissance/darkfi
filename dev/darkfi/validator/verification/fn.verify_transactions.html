Expand description

Verify a set of Transaction in sequence and apply them if all are valid.

In case any of the transactions fail, they will be returned to the caller as an error. If all transactions are valid, the function will return the total gas used and total paid fees from all the transactions. Additionally, their hash is appended to the provided Merkle tree.
