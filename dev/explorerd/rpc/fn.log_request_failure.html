fn log_request_failure(req_method: &str, params: &JsonValue, error: &JsonError)
Expand description

Auxiliary function that logs RPC request failures by generating a structured log message containing the provided req_method, params, and error details. Constructs a log target specific to the request method, formats the error message by stringifying the JSON parameters and error, and performs the log operation without returning a value.
